# 動詞變化

## 例外情況

一、看起來是上一段動詞，實際上是五段動詞的有：走る、切る、入る、交じる、要る、握る、齧る、散る、知る、千切る、参る、限る。

二、看起來是下一段動詞，實際上是五段動詞的有：照る、減る、茂る、滑る、耽る、帰る、捻る、蹴る、喋る。
